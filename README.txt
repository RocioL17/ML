## Este repositorio es para conectar Google Collab.

Utilice un ambiente virtual local con venv para correr los scripts.

En el caso que no encuentre el ambiente, presionar el botón de la derecha que dice: Kernel, y allí seleccionar el ambiente
virtual venv que creamos.

Se agrego un Script para que los outputs en HTML no se sobreescriban, y se guarden por versiones.

1. Comparamos Random Forest y XGBoost
2. Comparamos DT con XGBoost
3. Carpeta con ejercicios
